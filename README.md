# Filemanager

## :newspaper: About the project

A simple filemanager to automate file management tasks

### Content overview

    .
    ├── filemanager.py - main program and program logic
    ├── LICENSE - contains license text
    └── README.md - contains project information

## :notebook: Features

* Auto move files
* Get folder and file size
* Delete duplicate files

## :runner: Getting started

```python
python3 filemanager.py
```
